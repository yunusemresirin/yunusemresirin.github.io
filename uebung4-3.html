4.3 Fibonacci

Die Fibonacci-Funktion ist folgendermaßen implementiert: 

            
                function Fibonacci(n) {
                    let f_n = 1
                    let f_n_1 = 1
                    let f_n_backup;
                    
                    for(let x = 1; x <= n; x++) {
                        console.log(f_n)
                        if(x > 1) {
                            f_n_backup = f_n;
                            
                            f_n = f_n + f_n_1
                            
                            f_n_1 = f_n_backup
                        }
                    }
    
                    // return f_n -> um die letzte Ziffer auszugeben
                }

Daraus lässt sich schlussfolgern, dass

  • bei Integer die letzte große Ziffer die 78. Ziffer mit 8944394323791464 ist, da es kleiner Number.MAX_SAFE_INTEGER = 9007199254740991 ist.
  • bei Number die letzte große Ziffer die 1476. Ziffer mit 1.3069892237633987e+308 ist, da es kleiner Number.MAX_VALUE = 1.7976931348623157e+308 ist.
  • bei BigInt die größte Ziffer Infinity ist.
